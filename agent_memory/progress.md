# 当前任务进度

## 成功标准

- 点击“投喂一下”后不再卡死，尽量复刻 WCEhance 的微信内部赞赏扫码链路。
- 使用用户当前赞赏码作为内嵌图和安装资源。
- 使用用户提供的左下角图标，处理为透明 PNG。
- 版本号按用户规则从 `1.0.8` 开始递增；当前同步到 `1.0.9`，后续依次 `1.1.1`、`1.1.2`，跳过 `1.1.0`。
- 首次安装登录账号后，或已登录状态下用新签名包覆盖安装后，首次打开显示倒计时弹窗；倒计时结束后点击“已知晓”关闭弹窗并自动关注公众号，关注失败只提示失败。
- 插件底部胶囊点击“关注公众号”时直接自动关注；关注失败再跳转公众号资料页，让用户手动关注。

## 范围边界

- 只处理赞赏入口、菜单滚动性能、左下角图标资源、版本号和用户确认的公众号关注链路。
- 不重构其它插件管理、配置或隐私逻辑。
